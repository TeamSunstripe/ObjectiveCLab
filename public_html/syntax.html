チュートリアル

Objective-C／基本構文

if文、for文、switch文、while文、do-while文の構文です。
書き方を忘れた時にも便利です。

if構文

ifは条件に合致するかしないかを判定する処理を行います。

if構文

構文名
表題(表題をタップして説明/解説表示)

if構文
条件が合えば、処理を行う

書き方

// Objectice-C
// if構文
if(条件){
  // 条件=YESの場合の処理
}
if～else構文
条件が合えば、処理（１）を行い、それ以外は、処理（２）を行う

書き方

// Objectice-C
// if～else構文
if(条件){
  // 条件=YESの場合の処理
}else{
  // 条件=NOの場合の処理
}
if～else if構文
条件１が合えば、処理（１）を行い、条件２が合えば、処理（２）を行い、それ以外は、処理（３）を行う

書き方

// Objectice-C
// if～else if構文
if(条件１){
  // 条件１=YESの場合の処理
}else if(条件２){
  // 条件２=YESの場合の処理
}else{
  // 条件１、条件２いずれもNOの場合の処理
}

三項演算子構文

ifと同様に三項演算子構文は、条件分岐する処理を行います。

三項演算子構文

【三項演算子構文】
構文名
表題(表題をタップして説明/解説表示)

説明/解説

三項演算子構文
(条件)?＜条件=YESの場合の処理＞:＜条件=NOの場合の処理＞;

※上記「if～else構文」と同じ処理になります。
書き方

// Objectice-C
// aとbが同じなら「おなじ」違う時は「ちがう」を返す。
NSString *str = (a==b)?@"おなじ":@"ちがう";

for構文

forは指定された回数だけ処理をループ実行する処理を行います。

for構文

【for構文】
構文名
表題(表題をタップして説明/解説表示)

説明/解説

for構文
forは指定された回数だけ処理をループ実行します。

書き方

// Objectice-C
// for構文
for(変数初期値; ループ回数; 変数増分){
  // 処理
}

（例）

// Objectice-C
// for例文
// 例：iを0から99まで100ループさせる
for(int i=0; i<100; i++) {
  // 処理
}

for文には高速列挙というリスト型の件数分ループ処理を行うということもできます。

高速列挙構文
コレクションクラスから取り出す型を指定して、変数を取り出し、処理を高速で行います。

※上記「for構文」の高速処理になります。
書き方

// Objectice-C
for(取り出す変数の型 取り出す変数名 in コレクションクラス){
　 // 処理
}

（例）

// Objectice-C
// 例：変数arrayに格納されているデータを変数strとして取り出してNSLogに出力する
// （※arrayはNSArray型、格納されているデータはNSString型とする）
for (NSString *str in array) {
  NSLog(str);
}

switch構文

switchは与えられた値から処理を選択する判定処理です。
判定はbreakで抜けない限り、caseの上から１つずつ判定していきますので注意が必要です。

【switch構文】
構文名
表題(表題をタップして説明/解説表示)

説明/解説

switch構文
switchは与える変数の値を、caseにわけて、判定する処理を実行します。

書き方

// Objectice-C
// switch構文
switch(変数){
  case 値１:
    // 変数=値１の場合の処理
    break;

  case 値２:
    // 変数=値２の場合の処理
    break;

  default:
    // 上記の処理をbreakで抜けない限り常に処理
    break;
}

while構文

whileは条件がYESの間繰り返し実行を続けるルーチンです。
なので、条件がNOもしくは、breakで抜けるタイミングを作らないと永久ループとなってしまいます。

【while構文】
構文名
表題(表題をタップして説明/解説表示)

説明/解説

while構文
whileは条件がYESの間繰り返し実行を続けるルーチンする処理です

書き方

// Objectice-C
// while構文
while(条件){
  // 条件がYESの場合処理
}
do-while構文
do-whileは基本的にwhile文と同じですが、while文が１回も実行されない可能性があるのに対し、do-while文は最低１回は必ず実行されるという違いがあります。

書き方

// Objectice-C
// do-while構文
do{
  // 条件がYESの場合処理
}while(条件);

条件について

ANDは「&&」、ORは「||」、NOTは「!」を使用します。

条件名

    // 処理

a || b

書き方

// Objectice-C
// aまたはb
if (a || b) {
    // 処理
}
!a
aではない

書き方

// Objectice-C
// aではない
if (!a) {
    // 処理
}

定数

定数を定義するには「#define」を使用します。

// Objectice-C
// 定数CONSTを定義する
#define CONST @"TEISU"

#defineは、ヘッダーファイルのインポート句の下ぐらいに記述します。
