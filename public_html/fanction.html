チュートリアル

Objective-C／関数

Objective-Cで利用できる関数と定数です。
Objective-CはC言語の拡張版なのでC言語で使用できる関数はそのまま使用できます。

主要関数

ここでは、Objective-Cでプログラミングをする上での様々な関数をまとめてみました。

算術関数

【算術関数】
関数名
表題(表題をタップして説明/解説表示)

説明/解説

int rand()
乱数を発生させる

（例）

// Objectice-C
    srand(time(nil));　//乱数初期化
    int val = rand()%50;　//0～49までの間で乱数を発生
int abs(int a)
整数値aの絶対値を求める

（例）

// Objectice-C
int val = abs(-8);
→8

※値が浮動小数点型の場合はfabs関数を使用します。

double fabs(double a)
浮動小数値aの絶対値を求める

（例）

// Objectice-C
double val = fabs(-12.345);
→12.345

※値が整数型の場合はabs関数を使用します。

double floor(double a)
小数点以下切り捨てる

（例）

// Objectice-C
double val = floor(12.345);
→12.000
double ceil(double a);
小数点以下切り上げる

（例）

// Objectice-C
double val = ceil(12.345);
→13.000
double pow(double a, double b)
aのb乗を求める

（例）

// Objectice-C
double val = pow(2, 3);
→8
double sqrt(double a)
平方根を求める

（例）

// Objectice-C
double val = sqrt(2);
→1.41421356
